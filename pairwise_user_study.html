Pairwise User Study / 成对用户研究

Task Introduction / 任务说明:
This user study evaluates two compositional action generation tasks: temporal composition and spatial-temporal composition.
本用户研究评估两类组合动作生成任务：时序组合 和 时空组合。
Temporal Composition / 时序组合:
The generated motion should perform multiple actions sequentially along the timeline. The key requirement is whether the motion follows the correct temporal order and produces smooth transitions between actions.
时序组合要求人体动作按照时间顺序依次完成多个子动作，重点观察动作顺序是否正确，以及不同动作之间的过渡是否自然、连贯。
Spatial-temporal Composition / 时空组合:
The generated motion should perform multiple actions over time while also satisfying body-part constraints. The key requirement is whether the specified body parts perform the corresponding actions at the correct time intervals.
时空组合不仅要求多个动作按照时间顺序完成，还要求指定身体部位在对应时间段执行相应动作，重点观察动作、时间和身体部位约束是否一致。
Instruction / 评价说明:
For each task, please compare Video A and Video B according to three criteria: naturalness, body coordination, and semantic consistency with the given action description.
对于每个任务，请根据给定动作描述，对比 Video A 和 Video B，并从三个方面进行选择： 动作自然性、身体协调性 和 语义一致性。
Criteria Explanation / 评价指标解释:
Naturalness / 动作自然性: Whether the motion looks realistic, smooth, and physically plausible, without obvious jitter, freezing, foot sliding, or unnatural poses.
判断动作是否真实、流畅、符合人体运动规律，是否存在明显抖动、停滞、脚底滑动或不自然姿态。
Body Coordination / 身体协调性: Whether different body parts move coherently with each other, especially during simultaneous actions or transitions between actions.
判断不同身体部位之间的运动是否协调一致，尤其关注多个部位同时运动或动作切换时是否自然配合。
Semantic Consistency / 语义一致性: Whether the motion correctly matches the given action description, including the action type, temporal order, timing, and specified body parts.
判断生成动作是否符合给定动作描述，包括动作类别、时间顺序、动作发生时间以及指定身体部位是否正确。
